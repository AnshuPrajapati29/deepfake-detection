# Deepfake Detection using Transfer Learning

## Overview
Deepfake image classification using EfficientNetB0.

## Technologies
- Python
- TensorFlow
- OpenCV
- EfficientNetB0

## Dataset
Deepfake and Real Images Dataset (Kaggle)

## Results
Precision (class 0 / Fake): 0.76
Recall (class 0 / Fake): 0.81
F1-score (class 0 / Fake): 0.79
Precision (class 1 / Real): 0.79
Recall (class 1 / Real): 0.74
F1-score (class 1 / Real): 0.77
Validation Accuracy: 0.7777 (or approximately 77.77%)
